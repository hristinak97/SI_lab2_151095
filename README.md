1. Бројот на региони

2. Со формулата V(G) = E - N + 2, (E - вкупниот број на врски помеѓу јазлите, N - бројот на јазли)      
   V(G)=20-6+2=6

3. Со формулата P + 1 (P - бројот на предикатни јазли)
   V(G)=5+1=6
